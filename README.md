# 🚀 Goodychat （Local）

<br>

<br>

① リポジトリをクローン

```bash
git clone https://github.com/VioletBlend/goodychat.git
cd goodychat
```

② 必要なライブラリをインストール

```bash
pip install -r requirements.txt
```

（実行方法）

<br>

## 📦 必要なライブラリをインストール

<br>

```bash
pip install streamlit openai
```

<br>

<br>

🔑 OpenAI APIキーを設定（Windows）

<br>

```bash
setx OPENAI_API_KEY "ここにあなたのAPIキー"
```

<br>

※ 設定後はコマンドプロンプトを閉じて開き直してください。

<br>

▶️ アプリを実行

<br>

```bash
streamlit run app.py
```

<br>

ブラウザが自動で開き、アプリが起動します。
